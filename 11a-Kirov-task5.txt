>> function pl3d  -> създаваме функцията pl3d
z = 0 : pi/200 : 2 * pi; -> създаваме масиот от 0 до 2 * pi със стъпка pi / 200
x = cos(z); -> създаваме масив x, където са cos на ст-тите на z
y = sin(z); -> създаваме масив y, където са sin на ст-тите на z
plot3(x, y, z, "+"); -> чертае графиката със плюсове
end -> край на функцията
>> pl3d -> извикваме функцията
